# kreuzcrawl - FFI (C/C++) Bindings

[![Bindings by alef](https://img.shields.io/badge/bindings%20by-alef%20%D7%90-007ec6)](https://github.com/kreuzberg-dev/alef)

High-performance web crawling engine

## Installation

Link against `libkreuzcrawl_ffi` and include `kreuzcrawl.h`.

See the build instructions in the main repository.

```c
#include "kreuzcrawl.h"

    // See https://github.com/kreuzberg-dev/kreuzcrawl for usage examples.
    return 0;
}
```
