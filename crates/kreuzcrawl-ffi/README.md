# kreuzcrawl - FFI (C/C++) Bindings

High-performance web crawling engine

## Installation

Link against `libkreuzcrawl_ffi` and include `kreuzcrawl.h`.

See the build instructions in the main repository.

## Quick Start

```c
#include "kreuzcrawl.h"

int main(void) {
    // See https://github.com/xberg-io/kreuzcrawl for usage examples.
    return 0;
}
```
