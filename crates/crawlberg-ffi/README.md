# crawlberg - FFI (C/C++) Bindings

High-performance web crawling engine

## Installation

Link against `libcrawlberg_ffi` and include `crawlberg.h`.

See the build instructions in the main repository.

## Quick Start

```c
#include "crawlberg.h"

int main(void) {
    // See https://github.com/xberg-io/crawlberg for usage examples.
    return 0;
}
```
